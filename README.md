# SmartBite – MERN Restaurant Reservation & Online Ordering System

SmartBite is a full-stack MERN application designed to modernize restaurant operations.  
It allows users to browse menus, add food items to cart, order online, reserve dining tables, and submit feedback — while admins can manage menu items, orders, and reservations using a secure dashboard.

---

## Tech Stack Used
| Layer | Technology |
|-------|------------|
| Frontend | React.js, HTML, CSS, Bootstrap, JavaScript |
| Backend | Node.js, Express.js |
| Database | MongoDB (MongoDB Atlas) |
| Authentication | JWT (JSON Web Token) |
| Tools | Postman API Testing, GitHub, VS Code |
| Deployment | (Optional) Frontend – Vercel, Backend – Render |

---

## Features

### User Features
- View restaurant menu
- Add items to cart
- Online checkout & order placement
- Book table reservation (select date, time, seating count)
- Login & signup using JWT authentication
- View order history
- Submit feedback to restaurant

### Admin Features
- Manage menu items (Add / Update / Delete)
- View and manage user orders
- View table reservations
- Access feedback from customers
